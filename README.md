* :arrow_right: Add Playlist Favorite Song
* :arrow_right: Logout Account


## Installation

* Clone Project
```
https://github.com/Fahmiali99/callData-Spotify.git
```
* Go To Directory
```
cd callData-Spotify
```
* Set up Spotify ID
 : `KEY_API_ID` at .env file
* Start the server
```
npm start
```
